Installation¶

Stable release¶

To install Hotelling T2, run this command in your terminal:

$ pip install hotelling

This is the preferred method to install Hotelling T2, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Hotelling T2 can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/dionresearch/hotelling

Or download the tarball:

$ curl -OJL https://github.com/dionresearch/hotelling/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
